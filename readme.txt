Orbisius Resume Organizer allows you to organize resumes from job applicants.

= Features =
* Create *unlimited* resume entries
* Attachments: attach *unlimited* files and define their type e.g. Resume, Cover Letter, References etc.
* Files are grouped into *deep* folder structure e.g. /orbisius-resume-organizer/files/1/2/3/4/ so they can be loaded quickly.
* Quick and easy interface for deleting attachments (ajax).
* Easy to use interface
* TODO: Search

= Usage =

Go to Admin > Orbisius Resume Organizer.

Then Add a new resume click on Save Draft or Publish.
Then extra fields will show up and you can attach files.

== Installation ==

1. Unzip the package, and upload `orbisius-resume-organizer` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==
